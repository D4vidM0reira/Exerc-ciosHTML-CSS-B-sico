Flex container: Direction, wrap e Flow

Direction - controla a direção dos itens dentro do contêiner flexível.

  • "row" (horizontal/padrão)
  • "row-reverse" (horizontal inversa)
  • "column" (vertical)
  • "column-reverse" (vertical inversa)
