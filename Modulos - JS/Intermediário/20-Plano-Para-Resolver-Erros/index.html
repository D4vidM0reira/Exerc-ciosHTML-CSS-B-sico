Plano para resolver erros

Encontrando e resolvendo erros

  • Pesquisar o erro no Google e especificar a linguagem;
  • Sempre estudar para entender o erro;
  • Os erros vão aparecer no console;
  • O console especificará a linha e o arquivo onde ocorreu o erro;
  • Erro de escrita da variável: a variável não foi definida;
  • Função não definida; erro de escrita ou a função ainda não foi criada no JS;
  • No caso de erro, o script irá parar a execução;
  • O script deve ser colocado sempre no final do body. Caso contrário, o código será executado antes do DOM ser renderizado, e ele não encontrará algumas propriedades, resultando em retorno null;
  • Sempre entenda o código que está sendo copiado, pois caso ocorram outros erros, será mais fácil corrigi-los;
